脚本目的是自动收取自己和好友的蚂蚁森林能量
基于autoJS实现

requirements:
- OS:android

# g.png
用于在截屏的图片中找到有能量好友位置的匹配图片

# green.js
老版脚本，所有函数都在一个里面

# greenBasedOnCoordinate.js
脚本分为两种类型，这个是基于坐标的

# greenBasedOnWidget.js
这个是基于控件的，目前有问题，未开发完成

# job.js
定时执行greenBasedOnCoordinate.js的脚本

# pwd.js
用于解锁屏幕

# test.js
截屏给坐标截取g.png的
